# epitech-tests

## Installation
* cd [dossier où cloner le repo]
* git clone
* sudo mv [dossier clôné] /usr/bin
* sudo ln -s /usr/bin/epitech-tests/tek /usr/bin/tek
* cd /usr/bin/epitech-tests/
* sudo npm -g install nodeunit
* sudo npm install

## Utilisation
#### Tester
* cd [depot du jour (ex: "rendu/piscine_cpp_d07m/")]
* tek [jour (ex: "d07m")]

#### Update
* tek update
